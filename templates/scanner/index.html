🔐 파일 보안 스캐너

VirusTotal API 기반 실시간 악성코드 탐지 시스템

🛡️ 이 시스템은 무엇인가요?

본 시스템은 전 세계 75개 백신 엔진을 보유한 VirusTotal API를 활용하여 업로드된 파일의 악성코드 여부를 실시간으로 탐지합니다. 단순 확장자 기반 필터링이 아닌 SHA256 해시 기반 데이터 분석으로 높은 정확도의 보안 검사를 제공합니다.

🔑
SHA256 해시 분석
파일의 고유 지문을 계산하여 VT DB와 대조
🌐
75개 엔진 검사
전 세계 백신 엔진이 동시에 파일을 분석
🚨
자동 격리
악성 탐지 즉시 quarantine 폴더로 분리
👁️
실시간 감시
폴더 감시 중 새 파일 추가 시 자동 스캔

📖 사용 방법

1

VirusTotal 사이트에서 API 키 발급

2

아래 입력란에 API 키 입력

3

파일 선택 또는 폴더 경로 입력

4

결과 확인 — 악성 파일은 자동 격리

5

전체 기록은 대시보드에서 확인
